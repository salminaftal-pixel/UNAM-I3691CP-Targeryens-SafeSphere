
# UNAM-I3691CP-Targeryens-SafeSphere

Safe Sphere is a mobile safety awareness and prevention application designed to reduce accidents in mining and industrial work environments. The app provides safety education, training courses, quizzes, and progress tracking to help users prepare before entering hazardous workplaces.

It focuses on prevention rather than reaction by equipping workers, students, and visitors with the knowledge required to identify and avoid potential hazards.
